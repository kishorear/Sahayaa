4. TicketFormatterAgent
Purpose

Takes the newly created ticket’s metadata (ID and subject) plus the solution steps (coming from a call to the LLM), and assembles a final, well‐formatted professional ticket response (with email body) suitable for sending back to the user or ticketing system.
Responsibilities

    Receive Inputs

        Ticket ID: The ID assigned by the ticket service when a new ticket was created.

        Ticket Subject: A concise summary of the user’s issue (“VPN connectivity issue”).

        Solution Steps: A list or multi‐sentence explanation generated by the LLM describing how to resolve the issue.

    Apply the Formatting Template

        Using an LLM or a template engine, create a ticket body such as:

Ticket #<id> • <subject>

Dear Customer,

We have reviewed your request. Below are the steps to resolve your issue:

1. Step one…
2. Step two…
3. Step three…

If you need further assistance, please reply to this email.

Thank you,
Support Team

The agent uses an LLMTemplateTool with a template string like:

        "Ticket #{id} • {subject}\n\nDear Customer,\n\nBelow are the steps to resolve your issue:\n{steps}\n\nSincerely,\nSupport Team"

    Return Final Formatted String

        The agent’s output is a single string containing the fully formatted ticket, ready to be sent via email or inserted into the ticketing UI.

Tools

    LLMTemplateTool

        Template:

        "Ticket #{id} • {subject}\n\nDear Customer,\n\nBelow are the steps to resolve your issue:\n{steps}\n\nSincerely,\nSupport Team"

        The tool will substitute {id}, {subject}, and {steps} with the provided values.

Input / Output

    Input

{
  "id": 42,
  "subject": "VPN connectivity issue",
  "steps": "1. Restart your router.\n2. Reinstall the VPN client.\n3. Update VPN server address to vpn.example.com"
}

Processing Steps

    Populate the template with

        id = 42

        subject = "VPN connectivity issue"

        steps = "1. Restart your router...\n2. Reinstall the VPN client...\n3. Update VPN server..."

    Generate final string via LLMTemplateTool.

Output

{
  "output": "Ticket #42 • VPN connectivity issue\n\nDear Customer,\n\nBelow are the steps to resolve your issue:\n1. Restart your router.\n2. Reinstall the VPN client.\n3. Update VPN server address to vpn.example.com.\n\nSincerely,\nSupport Team"
}
