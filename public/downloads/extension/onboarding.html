Welcome to SupportAI Chat Widget

Thank you for installing our browser extension. Follow the steps below to set up the widget on your website.

1

Configure Your Widget

Open the extension popup by clicking on the SupportAI icon in your browser toolbar. Enter your Tenant ID and API Key from your SupportAI dashboard.

You can also customize the widget appearance by selecting a primary color and position.

2

Add Widget to a Webpage

Visit the website where you want to add the chat widget. Click on the SupportAI icon in your browser toolbar, then click the "Inject Widget" button.

The chat widget will appear on the page immediately. This is a temporary installation for testing purposes.

3

Permanent Installation

For a permanent installation, add the following script to your website's HTML:

<script src="https://supportai.com/widget.js?tenant=[YOUR_TENANT_ID]"></script>

Replace [YOUR_TENANT_ID] with your actual Tenant ID from your SupportAI dashboard.

4

Advanced Configuration

For advanced configuration, you can add this script before the widget script:

<script> window.supportAiConfig = { tenantId: "YOUR_TENANT_ID", apiKey: "YOUR_API_KEY", primaryColor: "#6366F1", position: "right", autoOpen: false, branding: true }; </script>
