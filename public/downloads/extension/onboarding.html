Welcome to SupportAI Widget

This extension allows you to easily add a customer support chat widget to any website you visit.

Getting Started

1

Configure Your Widget

Before you can use the widget, you need to configure it with your SupportAI credentials:

  • Click on the SupportAI extension icon in your browser toolbar
  • Enter your Tenant ID and API Key from your SupportAI dashboard
  • Customize the appearance and behavior settings if desired
2

Activate on Websites

You can add the chat widget to any website you browse:

  • Navigate to the website where you want to add the widget
  • Click on the SupportAI extension icon
  • Click the "Activate on This Site" button
  • The chat widget will appear in the corner of the website
3

Manage Your Widgets

You can manage which websites have the widget activated:

  • The widget will automatically appear on websites where you've activated it
  • To remove the widget from a website, click "Deactivate" in the extension popup
  • Your settings are saved separately for each website

For Website Owners

If you own a website and want to permanently add the SupportAI widget, we recommend using our standard installation method:

  1. Log in to your SupportAI dashboard
  2. Navigate to the Chat Widget page
  3. Copy the provided installation code
  4. Add the code to your website's HTML

This provides better integration and more advanced customization options.
