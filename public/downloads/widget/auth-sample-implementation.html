Integration Demonstration

This page demonstrates the Support AI Chat Widget with authentication integration. The widget will:

  1. Display a chat button in the bottom right corner
  2. When clicked, show a login/register form for users
  3. After authentication, connect to your configured AI provider
  4. Maintain the chat state as users browse your website

Try clicking the chat button to see the authentication flow in action!

Integration Code

Here's the simple code needed to add the Support AI Chat Widget with authentication to your website:

<script>
  // Widget configuration
  window.supportAiConfig = {
    tenantId: "__TENANT_ID__",
    apiKey: "__API_KEY__",
    primaryColor: "82AEEB",
    position: "right",
    apiEndpoint: "__API_ENDPOINT__",
    authEndpoint: "__AUTH_ENDPOINT__",
    requireAuth: true,
    greetingMessage: "Hello! Please log in to get assistance."
  };
</script>
<script src="auth-support-script.js" async></script>

Key Features

  • User Authentication - Login and registration built-in
  • Persistent Chat - Maintains conversation across page navigation
  • AI Provider Integration - Connects to your configured AI provider
  • Shadow DOM - Ensures the widget doesn't conflict with your styles
  • Responsive Design - Works great on all devices
  • Secure Token Handling - Authentication tokens stored securely

For more detailed integration instructions, see the auth-integration-guide.md file included in this package.
