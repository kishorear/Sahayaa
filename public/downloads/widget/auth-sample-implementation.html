Support AI Chat Widget Integration

A complete demonstration of the Support AI Chat Widget with authentication functionality

What's Included

This demo shows how to integrate the Support AI Chat Widget with user authentication into your website:

  • User Authentication: Require users to log in before chatting
  • API Key Integration: Connect to your tenant's AI configuration
  • Persistent Sessions: Remember logged-in users between page visits
  • Customizable Settings: Adjust appearance and behavior to match your site
  • Secure Communication: All data is transmitted securely

Configuration Options

Option Description Default
tenantId Your Support AI tenant ID Required
apiKey Your API key for authentication Required
primaryColor The primary color of the widget #6366F1
position Widget position (right, left, center) right
greetingMessage Initial message displayed in the chat How can I help you today?
requireAuth Whether users must log in before chatting true
autoOpen Whether to automatically open the chat widget false
branding Whether to show Support AI branding true
reportData Whether to send analytics data true

Integration Code

Add the following code to your website to integrate the Support AI Chat Widget:

<!-- Support AI Chat Widget Configuration --> <script> window.supportAiConfig = { tenantId: __TENANT_ID__, apiKey: "__API_KEY__", primaryColor: "__PRIMARY_COLOR__", position: "__POSITION__", greetingMessage: "__GREETING_MESSAGE__", requireAuth: true, autoOpen: __AUTO_OPEN__, branding: __BRANDING__, reportData: __REPORT_DATA__ }; </script> <!-- Support AI Chat Widget Script --> <script src="support-widget-auth.js" async></script>
