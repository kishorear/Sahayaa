Support AI Chat Widget Integration

This demo page demonstrates how to integrate the authenticatable Support AI chat widget into your website.

Quick Start

Add the following script to your website's HTML right before the closing </body> tag:

<!-- Support AI Chat Widget Configuration --> <script> window.supportAiConfig = { tenantId: __TENANT_ID__, apiKey: "__API_KEY__", primaryColor: "__PRIMARY_COLOR__", position: "__POSITION__", requireAuth: true, greetingMessage: "__GREETING_MESSAGE__", autoOpen: __AUTO_OPEN__, branding: __BRANDING__ }; </script> <!-- Support AI Chat Widget Script --> <script src="auth-widget.js" async></script>
Note: Replace the script src with the actual path to where you've placed the widget files.

Integration Options

Basic Integration

The simplest way to add the Support AI chat widget to your website:

<!-- Support AI Chat Widget (Basic Integration) --> <script src="auth-widget.js" async></script>

This approach uses the default configuration values specified when you downloaded this package.

Advanced Integration

Customize the widget appearance and behavior:

<!-- Support AI Chat Widget (Advanced Integration) --> <script> window.supportAiConfig = { // Required parameters tenantId: __TENANT_ID__, apiKey: "__API_KEY__", // Optional customizations primaryColor: "#4f46e5", // Hexadecimal color code (without #) position: "right", // Can be "left" or "right" requireAuth: true, // Whether to require user authentication greetingMessage: "Hello! Please log in to get support.", autoOpen: false, // Whether to automatically open the chat branding: true, // Whether to show "Powered by Support AI" serverUrl: "https://your-support-ai-instance.com" // Custom server URL if needed }; </script> <script src="auth-widget.js" async></script>

Programmatic Control

Control the widget programmatically from your own JavaScript code:

<!-- Support AI Chat Widget (Programmatic Integration) --> <script src="auth-widget.js" async></script> <script> // Wait for the widget to load window.addEventListener('load', function() { // Check if SupportAIChat is available if (window.SupportAIChat) { // Initialize with custom config window.SupportAIChat.init({ tenantId: __TENANT_ID__, apiKey: "__API_KEY__", requireAuth: true, primaryColor: "__PRIMARY_COLOR__" }); // Example: Open the chat when a button is clicked document.getElementById('open-support').addEventListener('click', function() { window.SupportAIChat.open(); }); // Example: Log out the current user document.getElementById('logout').addEventListener('click', function() { window.SupportAIChat.logout(); }); } }); </script>

Available Methods

  • SupportAIChat.init(config) - Initialize with custom configuration
  • SupportAIChat.open() - Open the chat window
  • SupportAIChat.close() - Close the chat window
  • SupportAIChat.logout() - Log out the current user

Authentication Flow

The widget provides built-in authentication with the following features:

  • User login with email and password
  • New user registration
  • Persistent login state using localStorage
  • Secure token-based authentication
Important: The authentication uses your Support AI tenant configuration. Users who register through the widget will be associated with your tenant. You can manage these users from your Support AI admin dashboard.

API Configuration

The widget automatically uses the AI provider configuration from your Support AI account settings. No additional setup is needed to connect to the configured AI providers.

Note: Changes to your AI provider configuration in the Support AI dashboard will be automatically reflected in the widget without requiring any code changes.
