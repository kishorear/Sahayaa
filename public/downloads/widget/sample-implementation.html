Integration Guide

To add the SAHAYAA.AI chat widget to your website, follow these steps:

Step 1: Add the Widget Script

Add the following code to your website, right before the closing </body> tag:

<!-- SAHAYAA.AI Chat Widget -->
<script>
  window.sahayaaAiConfig = {
    tenantId: __TENANT_ID__,
    apiKey: "__API_KEY__",
    primaryColor: "__PRIMARY_COLOR__",
    position: "__POSITION__",
    greetingMessage: "__GREETING_MESSAGE__",
    autoOpen: __AUTO_OPEN__,
    branding: __BRANDING__,
    reportData: __REPORT_DATA__,
    adminId: __ADMIN_ID__
  };
</script>
<script src="sahayaa-widget.js" async></script>

Step 2: Customize (Optional)

You can further customize the widget by modifying the configuration options above or by editing the CSS file (sahayaa-widget.css).

Step 3: Test

After adding the code, test the widget by navigating to your website and clicking the chat button.

Advanced Usage

For advanced use cases, you can programmatically control the widget using JavaScript:

// Open the chat window
SahayaaAIChat.open();

// Close the chat window
SahayaaAIChat.close();

// Send a message programmatically
SahayaaAIChat.sendMessage("Hello from code!");

// Listen for events
SahayaaAIChat.on('message_sent', function(data) {
  console.log('User sent a message:', data.content);
});
