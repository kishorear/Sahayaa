===================================================
    SupportAI Chat Widget - Windows Installation
===================================================

Thank you for downloading the SupportAI Chat Widget for Windows!

This installer package is compatible with:
- Windows 10 (all editions)
- Windows 11 (all editions)

Installation Instructions:
--------------------------
1. Extract all files from this ZIP archive to a folder on your computer
2. Right-click on the "SupportAIWidget_Setup.bat" file
3. Select "Run as administrator" from the context menu
4. Follow the on-screen instructions to complete the installation

The installer will:
- Create necessary program files in your Program Files directory
- Add shortcuts to your desktop and Start Menu
- Register the application for easy uninstallation via Control Panel

Troubleshooting:
---------------
If you encounter any issues during installation:

1. Make sure you're running the installer with administrative privileges
2. Temporarily disable any antivirus software that might be blocking the installation
3. Ensure your Windows version is up to date
4. Contact our support team at support@supportai.com for assistance

For more information, visit: https://supportai.com/help/widget-installation
