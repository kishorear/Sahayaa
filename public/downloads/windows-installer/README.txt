SupportAI Widget - Windows Installer
Version 1.0.0
===================================

Thank you for downloading the SupportAI Widget Windows Installer!

This package contains everything you need to install and use the SupportAI Chat Widget on your website.

## QUICK INSTALLATION INSTRUCTIONS

1. Double-click the SupportAIWidget_Setup.exe file to start the installation wizard
2. Follow the on-screen instructions to complete the installation
3. Launch the SupportAI Widget application from your Start menu or desktop
4. Configure your Tenant ID and API Key in the settings panel
5. Use the embedded browser integration or export the widget code for your website

## PACKAGE CONTENTS

- SupportAIWidget_Setup.exe - The main installer file
- installer.nsi - The NSIS installer script (for developer reference)
- README.txt - This file
- license.txt - License terms and conditions

## SYSTEM REQUIREMENTS

- Windows 8/10/11 (64-bit recommended)
- 100MB of free disk space
- Internet connection for widget functionality
- Any modern web browser (Chrome, Firefox, Edge recommended)

## CONFIGURATION

After installation, you will need to configure your widget with:

- Tenant ID: Your unique SupportAI tenant identifier
- API Key: Your SupportAI API authentication key
- Widget appearance settings (colors, position, etc.)

You can find these credentials in your SupportAI dashboard.

## USAGE OPTIONS

The SupportAI Widget can be used in two ways:

1. **Browser Extension Mode**: Automatically inject the widget into websites you browse
2. **Code Export Mode**: Generate and copy code to embed in your own website
