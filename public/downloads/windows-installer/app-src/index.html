Getting Started

To use the SupportAI Widget on your website, follow these steps:

  1. Configure your account settings with your SupportAI Tenant ID and API Key
  2. Customize the widget appearance to match your brand
  3. Copy the generated code or export it as a file
  4. Add the code to your website's HTML, right before the closing </body> tag
