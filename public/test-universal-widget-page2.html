Chat Widget Features:

Persistent Chat State

The chat widget maintains its state (open/closed) and conversation history as you navigate between pages.

Session Storage

Uses sessionStorage instead of localStorage to persist chat data during the browsing session.

Shadow DOM Isolation

The widget is isolated from the page's styling using Shadow DOM for reliable appearance.
